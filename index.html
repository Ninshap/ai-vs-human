Популярные и полезные искусственные интеллекты

Нейросети для работы с изображениями

  • MidJourney – генератор изображений по текстовым запросам.
    Использует искусственный интеллект для создания художественных изображений, отвечающих запросам пользователя.
  • Stable Diffusion – открытая нейросеть для генерации изображений.
    Открытый фреймворк для создания изображений, основанный на диффузионных моделях ИИ.
  • Leonardo AI – ИИ для генерации и редактирования изображений.
    Создаёт изображения на основе текстовых запросов и позволяет редактировать их после создания.
  • DALL-E 2 – генерация изображений из текста, созданная OpenAI.
    ИИ, который может создавать изображения по текстовому описанию, используя технологии глубокого обучения.
  • Шедеврум – генератор изображения и видео из текста, созданная Yandex.
    Проект, сервис и приложение с помощью которого пользователи могут генерировать изображения и видео по текстовому описанию.

Нейросети для работы с текстом

  • DeepSeek – поисковая система на основе ИИ.
    Использует ИИ для улучшения поиска информации в интернете, предлагая более точные и быстрые результаты.
  • ChatGPT – чат-бот, способный генерировать текст, вести диалог и выполнять задачи.
    Использует GPT для создания текстов, ответов на вопросы, обучения и выполнения различных задач.
  • Giga.Chat – генерация по запросу текстов и изображений и музыки.
    Чат-бот разработанный Сбером который способен генерировать по запросу тексты и изображения, писать программный код и создавать музыку.
  • GPT от Яндекса – ИИ для работы с текстовыми запросами.
    Модель ИИ от Яндекса, которая генерирует текст и решает различные задачи.
  • Microsoft Copilot – ИИ для работы с текстовыми запросами и генерацией изображений.
    Генеративный чат-бот ИИ, разработанный корпорацией Microsoft. Основанный на серии больших языковых моделей GPT-4.

Нейросеть для работы со звуком

  • Suno – ИИ для создания музыки.
    Платформа для генерации музыки с помощью ИИ.
  • Voice.ai – ИИ для изменения голоса, а также редактирования аудио файлов.
    Voice AI дает вам возможность изменять свой голос в режиме реального времени, а также удалять вокал из песни и разделять звуковой файл на несколько аудиодорожек.
