Популярные нейросети

Нейросети для работы с изображениями

  • MidJourney – генератор изображений по текстовым запросам.
    Использует искусственный интеллект для создания художественных изображений, отвечающих запросам пользователя.
  • Stable Diffusion – открытая нейросеть для генерации изображений.
    Открытый фреймворк для создания изображений, основанный на диффузионных моделях ИИ.
  • Leonardo AI – ИИ для генерации и редактирования изображений.
    Создаёт изображения на основе текстовых запросов и позволяет редактировать их после создания.
  • DALL-E 2 – генерация изображений из текста, созданная OpenAI.
    ИИ, который может создавать изображения по текстовому описанию, используя технологии глубокого обучения.
  • Шедеврум – генератор изображения и видео из текста, созданная Yandex.
    Проект, сервис и приложение с помощью которого пользователи могут генерировать изображения и видео по текстовому описанию.
